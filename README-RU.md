# simple-name-picker

**Версия 1.0.1** · Английская версия: [README.md](./README.md).

Небольшое веб-приложение на **Vite + React + TypeScript** для **случайного выбора имени** из каталога, **оценки** имён (нравится / не нравится) и **банов**, из-за которых имя пропускается, пока вы его не вернёте. Состояние **сохраняется в браузере** (`localStorage`). Список релизов — в [CHANGELOG.md](./CHANGELOG.md).

## Возможности

- **Выбор** подставляет случайное имя среди тех, кто **не в бане** и у кого **нет положительной** оценки (при балле больше нуля имя не участвует в случайном выборе, пока оценку не снизят или не сбросят в панели оценок).
- **Нравится** / **Не нравится** меняют сохранённую оценку; затем подставляется **другое подходящее случайное имя**, если возможно (не то, которое вы только что оценили).
- **Бан** убирает текущее имя из пула и сразу выбирает снова; **Вернуть** в списке отброшенных возвращает имя.
- Боковые панели **Отброшенные** и **Лучшие по оценке**: баны относительно текущего каталога и **ненулевые** оценки (положительные и отрицательные), сортировка по баллу, ограничение числа строк (см. `TOP_RATED_DISPLAY_LIMIT` в `namePickerState.ts`).
- **Редактировать** открывает **Список имён**: два сохранённых пресета (**женский** и **мужской** списки). Кнопки **Вставить женские (шаблон)** / **Вставить мужские (шаблон)** подставляют встроенные списки из `src/data/names.ts` (`FEMALE_NAMES`, `MALE_NAMES`); **Сохранить и применить** сохраняет выбранный слот, делает его активным и обновляет каталог (оценки и баны сбрасываются).
- **Сбросить** снова применяет **активный сохранённый** пресет к каталогу и очищает оценки и баны.

Строка под заголовком показывает число имён в каталоге и сколько из них **в бане среди этого каталога** — тот же набор, что в «Отброшенных». «Сиротские» записи банов для имён вне списка там не учитываются.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173).

## Сохранение данных

- **Каталог + оценки + баны:** `simple-name-picker:names` (см. `src/namesStorage.ts`).
- **Пресеты и активный набор:** `simple-name-picker:preset-names` — JSON `{ female: string[], male: string[], selected?: 'female' | 'male' }`. Старый формат с одним массивом строк при загрузке переносится в женский пресет (см. `src/presetNamesStorage.ts`).
- **Формат (ключ names):** `names` (массив строк), `ratings` (объект «имя → число»), `banned` (массив строк). Данные только как массив строк по-прежнему читаются как список имён с пустыми оценками и банами.
